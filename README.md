# Israeli Domain Blocker 🇵🇸

## Description

A comprehensive browser extension that blocks websites associated with Israeli entities as a form of digital activism to protest against the occupation regime. This extension has been completely redesigned with authentic Palestinian visual identity, expanded functionality, and advanced features for effective boycotting.

## ✨ Major Features

### 🚫 Advanced Blocking System
- **Massive Domain Database:** 250+ domains across multiple categories
  - 85+ Israeli companies and organizations
  - 75+ Pro-Israeli advocacy companies (Big Tech, SaaS, Social Media)

- **Smart Categorization:** 25+ categories for transparency and education

## 🚀 Installation Guide

### Method 1: Direct Installation
1. **Download** the extension files from this repository (Click "Code" → "Download ZIP")
2. **Extract** the ZIP file to a folder on your computer
3. **Open** your browser's Extensions page:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
   - Brave: `brave://extensions/`
4. **Enable** "Developer mode" (toggle in top-right corner)
5. **Click** "Load unpacked" and select the extracted folder
6. **Success!** The extension icon appears in your toolbar

### Method 2: Git Clone
```bash
git clone https://github.com/oussamakou/Israeli-Domain-Blocker.git
cd Israeli-Domain-Blocker
# Then follow steps 3-6 above
```

## 🔧 Development & Customization

### Adding New Domains
1. **Edit background.js:** Add to appropriate category in `preSelectedDomains`
2. **Include Evidence:** Add documentation links in comments
3. **Test Thoroughly:** Verify blocking works correctly
4. **Update Categories:** Add to categorization system if new category

### Customizing Alternatives
1. **Edit background.js:** Modify `alternativesDatabase` object
2. **Add New Categories:** Create new alternative categories
3. **Update Matching Logic:** Enhance domain-to-alternative matching

### UI Customization
1. **Colors:** Modify CSS custom properties in popup.html
2. **Layout:** Adjust container widths and responsive breakpoints
3. **Animations:** Customize transition timing and effects
4. **Typography:** Update font families and sizes
